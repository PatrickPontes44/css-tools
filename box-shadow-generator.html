What is the CSS Box-Shadow Property and How it Works?

Box-Shadow is a CSS property that allows you to add shadows to an element. The shadow effect is added by changing the following values: offset-x, offset-y, blur-radius, spread-radius and color. The shadow will follow the shape of the element, if you put a Border-radius of 50% in a square, it will become a circle, and it's shadow will be a circle as well. You are not limited to just adding one shadow. You can add more than one, to do this just separate your shadows with a comma.

Examples

box-shadow: offset-x offset-y blur-radius spread-radius color;

/* or */

box-shadow: 5px 5px 5px 5px rgba(0,0,0,1);

/* the result will be a box with the offset-x, offset-y, blur-radius, spread-radius defined as 5px (five pixels) and the color as black with an opacity of 1 (one). */
/* or just */

box-shadow: 1px 1px 5px 1px rgba(0,0,0,0.6);

/* the result will be a box with offset-x of 1px, offset-y of 1px, blur-radius of 5px, spread-radius of 1px, and black as shadow color with opacity of 0.6 */


box-shadow: 0px 0px 10px 0px rgba(0,0,0,1);
border-radius: 50%;


/* a square element with 50% of border-radius will become a circle. See how the it's shadow will be rendered. */


box-shadow: -10px 0px 10px 1px rgba(86, 204, 56, 0.6), 10px 0px 15px 1px rgba(40, 64, 172, 0.6);


/* this example show how to add two shadows to an element (green on the left side and blue on the right side) */
