What is the CSS Text-Shadow Property and How it Works?

Text Shadow is a CSS property that allows you to add shadows to a text element. The shadow effect is added by changing the following values: offset-x, offset-y, blur-radius, and color. You are not limited to just adding one shadow. You can add more than one, to do this just separate your shadows with a comma.

Examples

text-shadow: offset-x offset-y blur-radius color;

/* or */

text-shadow: 5px 5px 5px rgba(114,114,114);

/* the result will be a text with the offset-x, offset-y, blur-radius, defined as 5px (five pixels) and the color as medium gray with an opacity of 1 (one). */ TEXT SHADOW /* or you can do */

box-shadow: 0px 0px 10px rgba(196, 21, 143, 0.9);

/* the result will be a text with offset-x of 0px, offset-y of 0px, blur-radius of 10px, and pink/violet as shadow color with opacity of 0.9 */ TEXT SHADOW


box-shadow: -5px 5px 10px rgba(95, 0, 0, 0.7), 5px 1px 10px rgba(53, 233, 9, 0.7);

TEXT SHADOW
/* will result in a text with the left side with a darker red shadow and the right side with a medium green shadow. */
